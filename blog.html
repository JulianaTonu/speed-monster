Here Are Some Questions with Answers

1.Difference between Local storage and Session storage

Ans: Local Storage: stores data in browser without any expiration date.Data is not deleted when the browser is closed.Data is available for future session. Session Storage: stores data in browser for a session.Data is not deleted when the browser is closed.Data is not available for future session

2.Difference between Global scope and block scope

Ans: A javascript block scoped means that the variable defined within a block will not be accessible from outside the block.we can not access the let and const variables that we declare inside a block from outside a block
