Here Are Some Questions with Answers

1.Difference between Local storage and Session storage

Ans: Local Storage: stores data in browser without any expiration date.Data is not deleted when the browser is closed.Data is available for future session. Session Storage: stores data in browser for a session.Data is not deleted when the browser is closed.Data is not available for future session

2.Difference between Global scope and block scope

Ans: A javascript block scoped means that the variable defined within a block will not be accessible from outside the block.we can not access the let and const variables that we declare inside a block from outside a block

3.How does javascript event loop works?

Ans: The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks. If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.

4.How many ways we can get undefined?

Ans:undefined is a property of the global object.A variable that has not been assigned a value is of type undefined. A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned.
